# Requirement
Docker
docker-compose

# Usage
``` bash
$ git clone git@github.com:H0R15H0/dockerized-vue-template.git REPONAME
$ cd REPONAME
$ docker-compose build
$ docker-compose run app npm install
$ docker-compose up
```

# Open
```
localhost:8080/
```
